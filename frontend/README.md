# Agentic GeoAI Platform - Frontend

This frontend is a Vite + React + TypeScript application intended to be the dashboard for the Agentic GeoAI Platform.

Features:
- Firebase Authentication (Email + Google)
- React Router-based pages
- React Query for data fetching
- Leaflet map integration
- Axios service attaching Firebase ID Token to backend requests

Getting started:

1. Install dependencies

```bash
cd frontend
npm install
```

2. Create `.env` with Firebase and backend config (see `.env.example`)

3. Run dev server

```bash
npm run dev
```
