# Agentic GeoAI Platform

Full-stack Agentic GeoAI platform with a Python backend and a Vite frontend.

## Project structure

- `backend/` - API, agent services, tests, and Python dependencies.
- `frontend/` - Vite application, UI source, and frontend dependencies.

See `backend/README.md` and `frontend/README.md` for component-specific details.
